🔄 Rehitung Data Historis

Halaman ini akan menjalankan script SQL untuk menghitung ulang semua data kehadiran dengan logika perhitungan baru:

Logika Perhitungan Baru:
• Checkout sebelum 13:00 → Tidak ada pengurangan istirahat, tidak ada pengurangan Rp 6.000
• Checkout 13:00 atau lebih → Tidak ada pengurangan istirahat, ada pengurangan Rp 6.000
Perintah Curl Manual (Alternatif):

curl -X POST https://absen-brown.vercel.app/api/recalculate \
       -H "Content-Type: application/json" \
       -d '{}'
⚠️ Catatan Penting:
• Pastikan Anda di halaman yang sama dengan API (https://absen-brown.vercel.app)
• Script akan memproses semua data yang ada
• Hanya gunakan jika diperlukan untuk rehitung data historis
• Untuk rehitung berkala, gunakan Supabase Cron Jobs
